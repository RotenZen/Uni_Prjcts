# PeerPath

## 📌 Description  
PeerPath is a self-learning platform powered by community knowledge.  
Users who are skilled or experienced in a particular field can share structured learning paths, including the steps they followed and useful resource links.  

Other users can explore these paths, follow them, and interact through likes, comments, reviews, and reports.  
Each followed path is added to the user’s profile with step-by-step checkboxes and a progress bar to track learning progress.

---

## 🚀 Features  
- 📚 Share structured learning paths  
- 🔗 Attach resources (links, materials, references)  
- 👥 Follow learning paths created by others  
- ✅ Step-by-step progress tracking with checkboxes  
- 📊 Visual progress bar for completion tracking  
- 💬 Like, comment, review, and report system  
- 👤 Personalized user profiles  

## Admin Features
- Admins block links from irrelevant or foul websites
- Admins can review reports and take action
- Admins can delete posts.
